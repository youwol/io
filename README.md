# @youwol/io

## Documentation

See the [online documentation](https://youwol.github.io/io/dist/docs/index.html).

## Use

To install the required dependencies:

```shell
yarn
```

To build for development:

```shell
yarn build:dev
```

To build for production:

```shell
yarn build:prod
```

To test:

```shell
yarn test
```
